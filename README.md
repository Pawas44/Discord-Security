# Discord Bot - All-in-One

A comprehensive Discord bot with moderation, security, utility, verification, ticket, and community management systems.

## Features

### Moderation

* Ban / Kick / Mute system
* Auto moderation
* Channel locking
* Warning system
* Role management

### Security / Antinuke

* Anti Ban
* Anti Kick
* Anti Bot Add
* Anti Webhook
* Anti Channel Delete/Create
* Anti Role Delete/Create
* Panic mode

### Utility

* Verification system
* Welcome system
* Auto responder
* Sticky messages
* Logging system

### Voice & Community

* Join to Create VC
* Voice management
* Live leaderboard

### Ticket System

* Support tickets
* Staff management

---

# Installation

## Requirements

* Node.js 18+ or 20+
* MongoDB Atlas
* Discord Bot Token

---

## Install Dependencies

```bash
npm install
```

---

## Configuration

Edit `config.json`

```json
{
  "TOKEN": "YOUR_BOT_TOKEN",
  "MONGO_DB": "YOUR_MONGODB_URL",
  "PREFIX": "&"
}
```

---

## Start Bot

```bash
npm start
```
